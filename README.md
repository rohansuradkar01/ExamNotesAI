## ✨ Features

- **🤖 AI-Based Notes Generation – Generate detailed exam-ready notes instantly
- **📊 Auto Diagrams & Charts – AI creates visual learning aids
- **⚡ Revision Mode – Short & crisp notes for quick revision
- **💳 Credit-Based System – Pay-as-you-use model
- **🏷️ Tag Filtering** - Organize and filter by tags
- **🔐 Secure Authentication – JWT-based login system
- **📱 Responsive Design** - Works on desktop and mobile

🚀 Quick Start

Prerequisites
React.js
Node.js (v18 or higher)
Express.js
MongoDB (running locally on port 27017)
1. Clone the Repository
cd ExamNotesAI
2. Backend Setup
# Navigate to backend
cd server

# Install dependencies
npm install

# Start the server (development mode)
npm run dev
The server will run on http://localhost:5000

3. Frontend Setup
Open a new terminal:

# Navigate to frontend
cd client

# Install dependencies
npm install

# Start the development server
npm run dev
The client will run on http://localhost:3000
